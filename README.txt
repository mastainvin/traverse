
Bienvenue sur Traverse ! 

Pour utiliser Traverse : 

	1. Ouvrez le dossier correspondant à votre système 
	d'exploitation avec un Terminal.

	2. Utilisez la commande make (linux n'a pas pus 
	être tester)

	3. Rendez-vous dans le répertoire src

	4. Ouvrez l'executable "traverse" (traverse.exe 
	sur windows)

Pour supprimer les fichiers objet et l'executable : 
	
	1. Ouvrez le dossier correspondant à votre système 
	d'exploitation avec un Terminal.

	2. Utilisez la commande make clean pour les objets

	3. Utilisez la commande make cleanall pour les
	objets et l'executable.

Pour utiliser les sauvegardes :

	1. Les sauvegardes sont toutes disponibles dans le
	jeu.

	2. Cliquez sur charger une partie

	3. Trouvez la sauvegarde que vous cherchez avec 
	les flèches ou en cliquant sur les flèches pour 
	vous déplacer.

	4. Cliquez sur jouer pour lancer la partie.


	Le nom des sauvegardes est expliqué ici :

	condvictoire : vous permet de tester qu'un joueur a 
	bien gagné lorsque ses pions 	sont dans la zone 
	d'un autre joueur

	testtriangle : vous permet de tester les déplacements
	 simples du triangle

	testcarre : vous permet de tester les déplacements 
	simples du carré

	testlosange : vous permet de tester les déplacements 
	simples du losange

	testrond : vous permet de tester les déplacements 
	simples du rond

	zonenonlibre : vous permet de vérifier qu'un joueur a
	perdu lorsqu'il n'a pas libéré sazone de départ

	sautrondlosang : vous permet de vérifier les sauts
	 du rond et du losange

	sauttricarre : vous permet de vérifier les sauts du 
	carré et du triangle

	iagagne : permet de verifier que l'ia puisse gagner

	bordtriangle : permet de vérifier la règle de bords

	
	Si vous rencontrez des problèmes ou que les sauve-
	garde sont corrompus copiers le fichiers saves.txt
	dans le dossier data du jeu.

Pour lancer une nouvelle partie : 

	1. Cliquez sur nouvelle partie

	2. Choisissez le nombre de joueurs (1 ajoute auto-
	magiquement une IA), la difficulté si vous jouez 
	seul et si les tours sont chronométrés

	3. Cliquez sur commencer.

Pour jouer :

	1. Sur le pion que vous voulez déplacer (la case
	sera alors de la couleurs de votre joueurs)

	2. Les cases accessibles sont affichées en gris, 
	cliquez dessus pour vous déplacer. Vous pouvez 
	le faire tant que vous voulez et que vous pouvez 
	jouer.

	3. Si vous voulez terminer le tour cliquez sur la
	flèche en haut à droite.

	4. Si vous voulez quitter ou sauvegarder faites 
	echap ou cliquez sur la bouton pause en haut à
	gauche.

Pour modifier les options :

	1. Rendez-vous dans options

	2. Cochez ou décochez ce que vous voulez modifier

	3. Si vous avez modifier le jeu se fermera pour 
	valider les changements


Remarque sur l'IA :
	
	L'IA est totalement fonctionnel et est capable de
	vous battre ! 

	Cependant, la consommation de la mémoire est très
	élever surtout quand le joueur est rapide.

	Ceci est dû au nombre de possibilités enregistrées

Bon jeu !
